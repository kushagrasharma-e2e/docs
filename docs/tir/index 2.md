# TIR

TIR (Train, Inference, Research) is E2E Networks' intelligence and research platform for AI/ML workloads.

## Platform Overview

TIR provides infrastructure for:

- **Training** - Train machine learning models at scale
- **Inference** - Deploy and serve models for predictions
- **Research** - Experiment and iterate on AI projects
