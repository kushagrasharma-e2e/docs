# TIR Quickstart

Use this page when you want the shortest route from the docs homepage into a relevant local TIR path.

## Recommended flow

1. Decide whether your immediate job is **training**, **inference**, or **research**.
2. Open the matching workflow page in this curated section so you start from the right intent.
3. Keep [Billing and usage](../guides/billing-and-usage.md) nearby if cost visibility matters to the task.
4. Continue to the [official TIR docs home](https://docs.e2enetworks.com/docs/tir/) only when the local route intentionally stops short of the detail you need.
