# TIR Overview

Use the TIR section when the job is centered on AI and ML workloads.

This section is organized around:

- **Getting Started** for fast orientation
- **Workflows** for training, inference, and research paths
- **Guides** for platform navigation and usage visibility
- **Troubleshooting** for support and release-awareness
- **Reference** for a concise map of the section

## Start here

- If you want the fastest path in, go to [Quickstart](quickstart.md).
- If you know the workload type already, jump straight to [Training](../workflows/training.md), [Inference](../workflows/inference.md), or [Research](../workflows/research.md).
- If you need the broader live product docs, use the [official TIR documentation home](https://docs.e2enetworks.com/docs/tir/).
