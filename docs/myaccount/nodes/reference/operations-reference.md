# Operations Reference

This page is a quick lookup for the main node actions.

## Node actions

- `Power Off`
- `Start`
- `Reboot`
- `Save Image`
- `Move To Another Project`
- `Convert to Committed Node`
- `Lock Node`

## Volume actions

- `Attach`
- `Detach`
- `Create Snapshot`
- `Convert to Committed`
- `Upgrade Volume`

## Snapshot actions

- `Create Snapshot`
- `Save as Image`
- `Lock`
- `Unlock`
- `Delete`
- `Schedule Snapshot`
- `Disable Schedule`
- `Snapshot Lifecycle`

## Saved image actions

- `Rename`
- `Move To Another Project`
- `Export To Object Storage`
- `Delete`
- `Create Node`
- `Create Scale Group`

## Alert actions

- `Create`
- `Pause`
- `Resume`

## Bulk actions

Bulk actions are available for up to 10 nodes:

- `Power off`
- `Power on`
- `Reboot`
- `Lock`
- `Unlock`
