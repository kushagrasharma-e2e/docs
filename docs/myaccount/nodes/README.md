# Nodes

Nodes are one area inside the broader **Compute** section of MyAccount.

This documentation set is for the moment when you are not asking "How is this
product organized internally?" but "How do I get a node working for my own
goal?"

Use these docs if you want to:

- launch your first node from MyAccount
- connect to Linux or Windows successfully
- understand billing and plan choices before you commit
- take snapshots, save images, or move workloads between regions
- import your own image or migrate a VM from AWS
- add monitoring, alerts, encryption, and network controls
- troubleshoot access, disk, site, or monitoring problems
